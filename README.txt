============================================
ProContentCook - Standalone Version
============================================

Este é o site completo em HTML/CSS/JS puro, pronto para rodar no VSCode ou qualquer navegador.

============================================
COMO USAR:
============================================

1. Copie toda a pasta "standalone" para o seu computador

2. Crie uma pasta chamada "images" dentro da pasta standalone

3. Copie as seguintes imagens da pasta "src/assets/" para "standalone/images/":
   - hero-product.webp
   - bundle-preview.webp
   - recipe-card-feature.webp
   - recipe-feature-2.webp
   - step1-access.webp
   - step2-branding.webp
   - step3-share.webp
   - recipe-breakfast.webp
   - recipe-lunch.webp
   - recipe-airfryer.webp
   - recipe-salad-new.webp
   - recipe-smoothie.webp
   - recipe-main.webp

4. Abra o arquivo "index.html" no navegador
   - No VSCode: clique com botão direito > "Open with Live Server"
   - Ou simplesmente arraste o arquivo para o navegador
